In order to excute the code, make sure the following libraries are installed on system including common ML and Data Analysis libraries:
1. Streamlit
2. Pandas Profiling
3. Xgboost and Catboost

Furthermore, in order to run the streamlit dashboard. Perform the following steps:
1. Navigate to your streamlit_dashboard.py file directory
2. Open the terminal and type: stream run streamlit_dashboard.py
3. Hit Enter
